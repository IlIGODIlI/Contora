# Security Policy

## Supported Versions

Only the latest release version of Contora is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT create a public issue. Instead, report it privately to our security team.

Please email details of the vulnerability to: **security@contora.app**

Include the following details in your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue (proof-of-concept scripts or screenshots/scenarios are highly appreciated).
- Impact assessment and how this exploit affects users.

We will review your submission and respond within 48 hours to coordinate a patch and disclosure plan.
