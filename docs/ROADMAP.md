# Project Roadmap

This document outlines the planned future features and updates for Contora.

---

## Phase 1: On-Device Persistence (Q3 2026)
- **Local History Dashboard**: Save generated content strategies to the browser's IndexedDB so creators can access previous plans without re-running the quiz.
- **Bookmark System**: Allow creators to bookmark specific content ideas and hook audits to build their own local swipe files.
- **Export Formats**: Support downloading the full content strategy PDF or exporting directly to Notion databases.

## Phase 2: Creator Studio Integrations (Q4 2026)
- **Calendar Scheduler**: A drag-and-drop calendar view to map out weekly posting days directly.
- **Script Writing Wizard**: A simple markdown editor that automatically checks script drafts against hook audits and pacing rules in real-time.
- **Custom Niche Training**: Train custom local LLM prompts that study user-submitted reference documents (like brand guidelines or past high-performing scripts).

## Phase 3: Analytics Feedback Loop (Q1 2027)
- **Analytics Sync**: Securely connect to Instagram/TikTok Creator Graph APIs to import real post reach and completion rate metrics.
- **Self-Improving Recommendations**: Feed actual view metrics back into the strategy algorithm to automatically refine optimal posting schedules and content topics.
